# ARMAN — лендинг

**Клиент:** Мухтар
**Продукт:** ARMAN — ИИ-наставник по саморазвитию для русскоязычной молодёжи (18-28, КЗ + СНГ)
**Цель лендинга:** тест маркетингового канала, сбор регистраций. Не продуктовая валидация.
**Статус:** концепция и копирайт готовы, идём в Claude Design.

## Что в папке

- `source-spec.md` — оригинальная спека от Мухтара (v0.1)
- `brief.md` — финальный бриф для Claude Design. Это то что копируется в чат с Claude Design на старте проекта.
- `image-prompts.md` — отдельно собранные промты для иллюстраций (для прогона через Sora/MJ/DALL-E когда дойдёт до них)

## Решения зафиксированные на этом этапе

- В MVP будет минимальная геймификация: ИИ опрашивает, выставляет ранг, ведёт по методологии семи сфер. Поэтому лендинг не врёт продукту.
- Стиль: киберпанк-аниме, Казахстан будущего. Без вкраплений казахских слов в копирайт — аудитория шире КЗ.
- Видео-демо вырезано из первой итерации.
- Телефон в форме регистрации убран. Только ник + email.
- Все картинки кроме карточки персонажа на экране 3 — плейсхолдеры с описанием. Карточку персонажа Claude Design рисует реально в коде.
- Добавлен экран про экспертов и методологию + блок про разблокировку личного созвона как награду за прокачку.

## Что отложено

- Что показывать пользователю после регистрации (мягкий путь vs резкий путь в чат)
- Канон Армана: один герой для всех или у каждого свой
- Сценарий видео-демо (когда вернёмся к нему)
- Финальные имена и фото экспертов для блока на экране 5 — заполнит Мухтар
- Telegram-канал — есть готовый или нужно создать с нуля
- Домен — пока деплоим на Vercel с тем что есть
